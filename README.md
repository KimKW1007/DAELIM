# DAELIM Clone

### 사용 기술
 - HTML/CSS
 - jQuery

### 느낀 점
 - 반응형이 안 되어있는 페이지를 반응형으로 만드려고 하니 디자인 측면에서 어려웠지만 최대한 기본 디자인을 그대로 옮겼습니다.
 - 코딩을 배우고 얼마 안 지나 만들어서 조금 더 잘 만들 수 있을 것 같았는데 조급한 마음에 최대한 빠르게 만든 것 같습니다.
 - 슬라이드를 라이브러리 안쓰고 만들었기 때문에 조금 어색한 감이 있지만 그래도 선방한 것 같습니다.
 - 최대 1920px 에 맞춘 사이트라 넘어가면 깨지는 부분이 있기 때문에 추후 리펙토링을 해야할 것 같습니다.
